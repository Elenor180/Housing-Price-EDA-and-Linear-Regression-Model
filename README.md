# Housing Prices EDA - Machine Learning Price Predictor Model
##### By Randy Kubheka

## Overview
This project performs an Exploratory Data Analysis (EDA) on a Housing Prices dataset to identify key factors influencing property values. It includes data cleaning, visualization of trends, and recommendations for future modeling.

## Requirements
To run the notebook, ensure you have the following Python libraries installed:
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

## Setup Instructions
1. Place the `train.csv` dataset in the same directory as the notebook.
2. Open the terminal and run: `pip install pandas numpy matplotlib seaborn scikit-learn`
3. Launch Jupyter Notebook and run `Sybrin_ML_Intern_Assessment.ipynb`.
